# 行为准则

## 我们的承诺

为了营造一个开放和包容的环境，我们作为贡献者和维护者承诺：无论年龄、体型、可见或不可见的残疾、种族、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、人种、宗教或性向、性认同，参与本项目和社区的每一个人都不应受到骚扰。

## 我们的标准

营造积极环境的行为：
- 使用包容性的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注社区利益
- 同理心对待社区成员

不可接受的行为：
- 使用性化的语言或图像
- 钓鱼、侮辱或贬低性的评论及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息
- 其他在职业场合明显不当的行为

## 执行

如有不当行为，请联系项目维护者：[jiangxufa110@gmail.com]。所有投诉将被审查和调查，并以维护者认为必要和适当的方式作出回应。
