# Landing Page 示例

`index.html` 是 vibe-blocks 的官方介绍页面，展示积木体系的核心理念。

## 用途

- 团队内部分享时使用
- 部署到 GitHub Pages 作为项目主页
- 改造为你自己项目的积木体系介绍页

## 部署到 GitHub Pages

```bash
# 复制到 docs/ 目录或建立 gh-pages 分支
cp examples/landing-page/index.html docs/index.html
```

然后在仓库 Settings → Pages 选择 `main` 分支的 `/docs` 目录作为站点源。
