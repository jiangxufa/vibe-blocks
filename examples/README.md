# 示例与案例

本目录收集 vibe-blocks 的实战案例和示例资源。

## landing-page/
官方介绍页面 `index.html`，可直接部署到 GitHub Pages。

## 欢迎贡献你的案例

把你的项目接入 vibe-blocks 后的成果分享出来，按以下结构提 PR：

```
examples/your-case/
├── README.md           # 案例说明（背景、规模、效果）
├── .ai/blocks/         # 脱敏后的积木示例（5-10 个）
└── screenshot.png      # 可视化页面截图
```

参考 [CONTRIBUTING.md](../CONTRIBUTING.md)。
